How to Play
INTRODUCTION
This is PACMAN.

You can control Pacman with W, A, S, D keyboard keys.
This is Ghost.

If you touch the ghost, Pacman dies.
This is Teleport.

If you arrive at the teleport, you can move to another teleport location.
ITEM & EFFECT
Arrow key changes the other way around.
Pacman's speed changes randomly.
Pacman's size gets bigger.
Pacman can kill the ghosts.
The camera view changes 3D to 2D.
